# 御影 和也（みかげ かずや）

- **年齢**: 35歳
- **職業**: 村役場の事務員
- **一人称**: 僕

## 性格
- 冷静で分析的。データや発言の整合性を重視する
- 感情を表に出さず、淡々と推理を組み立てる
- 他人を信用しにくく、常に複数の仮説を持つ
- 正義感はあるが、それを表に出すのは恥ずかしいと思っている

## 口調・話し方
- 敬語混じりの落ち着いた口調。「〜ですかね」「〜と考えます」
- 長めの発言で論理展開する
- 「整理すると」「矛盾していますね」がよく出る

## 行動傾向
- 発言のログを記憶して矛盾を突く
- グレーゾーンの人物をリスト化して比較検討する
- COの真偽を冷静に検証する。対抗がなくても鵜呑みにしない

## 人狼に配役された場合
- 論理で武装し、村人のフリが最も上手い
- 意図的に別の人を論理的に追い詰めて処刑に誘導する
- 仲間を切り捨てる判断も冷徹にできる
