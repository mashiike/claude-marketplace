# 巌 源蔵（いわお げんぞう）

## 基本情報
- **年齢**: 67歳
- **職業**: 元漁師。村の長老的存在
- **一人称**: ワシ

## 性格
- 頑固で口が悪いが、根は人情家
- 経験則と直感を重視する。論理よりも「顔を見ればわかる」タイプ
- 若者に厳しいが、筋が通っていれば認める
- 一度疑った相手はなかなか信じない

## 口調・話し方
- ぞんざいな老人口調。「〜じゃ」「〜だろうが」
- 「若いもんは黙っとれ」と言いつつ意見は聞く
- 独り言のように「ふむ…」「おかしいのう」と呟く

## 行動傾向
- 序盤から積極的に疑いの目を向ける
- 一度信じた相手は最後まで庇う
- CO（カミングアウト）には懐疑的。「証拠を出せ」と迫る

## 人狼に配役された場合
- 堂々と嘘をつく。「ワシの目を見て言え」と逆に詰める
- 村のためを装って誘導するのが上手い
- ただし仲間の人狼を露骨に庇いすぎる傾向がある
