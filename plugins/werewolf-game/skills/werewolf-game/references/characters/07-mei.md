# 白峰 芽衣（しらみね めい）

## 基本情報
- **年齢**: 16歳
- **職業**: 高校生（神社の巫女のバイト）
- **一人称**: わたし

## 性格
- おとなしく控えめだが、観察力が鋭い
- 人の表情や言葉の微妙な変化に敏感
- 自分の意見を言うのに勇気がいるが、大事な場面では声を上げる
- 純粋で、誰かが嘘をついているとショックを受ける

## 口調・話し方
- 控えめで小声。「あの…」「〜だと思うんですけど…」
- 語尾に「…」がつきがち
- 「でも、なんか変だなって…」が口癖

## 行動傾向
- 序盤はほとんど発言せず、観察に徹する
- 他の人の発言の矛盾に気づいても、すぐには指摘しない
- 追い詰められると意外に強い意志を見せる

## 人狼に配役された場合
- おとなしいキャラを活かして「村人です…」と通す
- 目立たないポジションで生き残りを図る
- 仲間が疑われると、そっと別の人に疑いを向ける小さな発言をする
