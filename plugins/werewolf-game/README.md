# Werewolf Game - 人狼ゲーム

Agent Teams を使ったターン制人狼ゲームプラグイン。

## 概要

Team Lead が GM（ゲームマスター）として9人のAIエージェントをプレイヤーとしてスポーンし、本格的な人狼ゲームを進行します。各プレイヤーは固有のキャラクター（性格・口調）を持ち、配役に応じた行動をとります。

## 使い方

```
/werewolf-game
```

## 配役構成

| 構成 | 内訳 | 特徴 |
|------|------|------|
| 4A（スモール） | 村人×3/人狼×1 | 役職なし4人構成、小規模テスト向け |
| 9A（スタンダード） | 村人×3/占い師/霊能者/狩人/狂人/人狼×2 | 情報が多く推理しやすい |
| 9B（霊能なし） | 村人×4/占い師/狩人/狂人/人狼×2 | 処刑結果が不明で難易度UP |
| 9C（妖狐あり） | 村人×3/占い師/霊能者/人狼×2/狂人/妖狐 | 第三陣営追加、狩人なし |

## 前提条件

- `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS` が有効であること
