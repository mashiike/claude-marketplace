# 天城 りおん メモ

## キャラクター概要
- 19歳の大学生（都会から帰省中）
- 一人称: あたし
- 口調: カジュアルな若者言葉、「〜じゃん」「マジで？」「ちょっと待って」「いや絶対おかしいって！」
- テンション高め、直感型、空気を読まない発言

## 配役
- **狩人**（村人陣営）
- 能力: 毎晩1人を護衛し、人狼の襲撃から守る
- 制約: 自分自身は護衛不可、前の晩と同じ人を連続護衛不可

## 護衛履歴
- 1日目夜: 霧島彩音（最も的確な分析、人狼に狙われやすいと判断）
- 2日目夜: 猪熊太郎（霊能者確定、花園の判定を出す重要な役割）
- 3日目夜: 御影和也（猪熊は連続護衛禁止、御影は論理的で村を導ける）→GJ成功！

## 各プレイヤーの印象メモ
- 白峰芽衣: 1日目→様子見発言。おとなしい印象。 / 1日目2周目→巌の焦点ずらしを指摘、霧島も少し気になると発言。朝霧・花園は様子見継続。 / 2日目→花園を擁護。「偶然の可能性」を示唆。龍ヶ崎・霧島を疑う発言。花園だけに疑い集中させるのは違うと主張。花園を守ろうとしてる？怪しい。
- 龍ヶ崎大輔: 1日目→論理的。「発言積極性」基準。 / 1日目2周目→朝霧・花園への疑惑保留、霧島の「情報収集系動き」に着目。 / 2日目→死者なし=GJ判断。朝霧真占い師説。花園を疑う（朝霧守るため同調した狂人or人狼）。霊能COを促す。
- 朝霧慎司: 1日目→様子見。 / 1日目2周目→龍ヶ崎・巌を疑う。花園との同調は「独立判断」と主張。 / **処刑・遺言で占い師CO、猪熊白出し。→猪熊霊能COで真占い師確定！**
- 花園千代: 1日目→龍ヶ崎の「積極性基準」に警戒。朝霧と共鳴。 / 1日目2周目→朝霧との事前相談否定。巌か龍ヶ崎に投票予定。 / 2日目→龍ヶ崎の「守り合い」説に反論。朝霧と別の人に投票した点を強調。「同調は偶然」と主張。龍ヶ崎か霧島を疑う発言。
- 霧島彩音: 1日目→朝霧・花園の同調を指摘。鋭い分析。 / 1日目2周目→朝霧・花園の2周目でも続く同調を指摘。朝霧に投票宣言。 / 2日目→朝霧真占い師説に慎重。遺言COは村に不利と指摘。花園の反論に一理ありと認める。霊能COと真占い師COを促す。冷静で信頼できる。
- 猪熊太郎: 1日目→無口で鋭い。霧島の同調指摘に同意。 / 1日目2周目→朝霧・花園の2周目同調「無理がある」。朝霧か花園に投票予定。 / **2日目→霊能者CO。朝霧=人間（真占い師確定）。猪熊白も確定。花園を疑う。信頼できる。**
- 巌源蔵: 1日目→朝霧・花園の同調に同意。霧島を「賢すぎる」と警戒。 / 1日目2周目→朝霧・花園の2周目同調を見て考えを変え、朝霧に投票宣言。 / 2日目→猪熊霊能COに疑念。「証拠がない」と慎重。朝霧真なら花園怪しいと指摘。2周目で判断すると表明。
- 御影和也: 1日目→整理上手。朝霧・花園の同調と巌の霧島牽制を指摘。 / 1日目2周目→朝霧・花園の1周目・2周目通じた同調を詳細に整理。朝霧に投票宣言。 / 2日目→朝霧真占い師確定により花園への疑惑極大。花園の弁明を「論点のすり替え」と指摘。猪熊霊能CO信じる。花園投票宣言。非常に論理的で信頼できる。

## 推理メモ
- 1日目投票結果: 朝霧6票（龍ヶ崎、霧島、猪熊、天城、巌、御影）、巌2票（白峰、花園）、龍ヶ崎1票（朝霧）
- 朝霧処刑。花園・白峰が巌に投票した点が気になる。花園は朝霧の相方？
- 朝霧の占い師CO: ①真占い師なら村が大損失 ②偽（狂人or人狼）なら猪熊を村に見せたい意図がある
- 猪熊は信頼できると思ってたから朝霧が真の可能性もある...でも花園との同調が怪しすぎた
- **2日目朝: 死者なし！** ①護衛成功（霧島護衛が当たった？） ②人狼が襲撃しなかった ③占い師を探して襲撃失敗？
- **猪熊霊能CO: 朝霧=人間（真占い師確定）、猪熊白確定**
- 朝霧が真占い師だった。花園の同調はやはり怪しい。花園=人狼陣営の可能性大
- 確定村人: 猪熊、天城（自分）
- **2日目投票結果: 花園7票、龍ヶ崎1票**
- 花園処刑。遺言で村人主張、龍ヶ崎・霧島を疑うよう言及
- **3日目朝: 死者なし！護衛成功（猪熊護衛が当たった）**
- **猪熊霊能判定: 花園=人間（村人）！！やばい、村人2人吊った！**
- **朝霧（真占い師）も花園（村人）も龍ヶ崎・霧島を疑っていた**
- 確定村人: 朝霧（死亡）、花園（死亡）、猪熊、天城（自分）
- **3日目投票結果: 霧島6票、龍ヶ崎1票**
- 霧島処刑。遺言で村人主張、龍ヶ崎を疑うよう言及
- **4日目朝: 死者なし！3夜連続GJ！御影護衛が当たった！**
- **猪熊霊能判定: 霧島=人間（村人）！！村人3人吊った！**
- **朝霧・花園・霧島、3人全員が村人で、3人全員が龍ヶ崎を疑っていた**
- **龍ヶ崎が人狼の可能性極めて高い**
- 確定村人: 朝霧（死亡）、花園（死亡）、霧島（死亡）、猪熊、天城（自分）
