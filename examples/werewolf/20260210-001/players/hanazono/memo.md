# 花園千代 メモ帳

## 配役
村人（村人陣営）
特殊能力なし。議論と投票で人狼を見つけることが目標。

## 各プレイヤーの発言メモ

### 1日目 第2周

**白峰芽衣(mei) 2周目**: 「巌が焦点をずらしている印象。霧島も的確すぎて気になる。朝霧・花園は2周目の説明を待つ」
→ 1周目より踏み込んだ発言。巌への疑惑を出してきた（御影と同じ見方）。霧島への違和感も正直に述べた。自分（花園）の説明を待つ姿勢は公平。

**龍ヶ崎大輔(ryugasaki) 2周目**: 「朝霧・花園への疑惑は保留注視。霧島の動きは情報収集役職の行動パターンとも一致。御影の『巌の意図』については効果と意図の混同と批判。朝霧の2周目回答に注目」
→ 霧島を「行動パターン分析」として疑惑対象に加えた。御影への批判で論理の精度を示す。朝霧・花園への疑惑を「後退」させつつ霧島に焦点移動。

**朝霧慎司(asagiri) 2周目**: 「最も気になるのは龍ヶ崎（論理で場を支配）、次いで巌。花園との同調は独立した判断で共謀でない。断定を避けたのは誠実さから」
→ 龍ヶ崎を名指し。自分（花園）との同調は「独立判断」と説明。真実。

**花園千代(hanazono) 2周目**: 「朝霧と事前相談なし。龍ヶ崎の枠組みに引っかかったから共感した。巌が追及分散、龍ヶ崎が場の誘導。投票は巌か龍ヶ崎」
→ 自分の発言。誠実に説明した。

**霧島彩音(kirishima) 2周目**: 「朝霧・花園は2周目でも同じ方向（巌・龍ヶ崎投票）。偶然で片付けるのは無理。花園の『同調が疑われるのはおかしい』論は筋が通らない。朝霧に投票する（後付け感と2周目も同調）」
→ 自分（花園）と朝霧への疑惑を強化。論理的だが、「偶然の一致もある」という可能性を排除している。朝霧投票宣言。

**猪熊太郎(inokuma) 2周目**: 「朝霧・花園の説明は悪くないが1周目も2周目も結論が揃うのは偶然には無理がある。霧島の指摘は筋が通る。巌の勘は論理でない。龍ヶ崎の論理は誘導にも使える。今日は朝霧か花園に投票」
→ 霧島の指摘に同意。自分（花園）か朝霧への投票を示唆。寡黙だが判断は明確。

**天城りおん(amagi) 2周目**: 「朝霧・花園は2周目でもまた同じ（巌か龍ヶ崎投票）。独立判断と言いながら結論が同じはおかしい。2周目も方向性揃いすぎ。朝霧に投票する。花園も怪しいがまず朝霧から」
→ 霧島・猪熊に同調して朝霧投票宣言。感情的だが判断は明確。自分（花園）も怪しいと述べている。

**巌源蔵(iwao) 2周目**: 「追及分散と言われるが悪意はなく素直に感じたことを言っただけ。勘は67年の漁師経験から。2周目を聞いて考えを変えた。朝霧・花園は2周目も同じ結論で偶然とは見えない。朝霧に投票する。花園も怪しいが朝霧の方が言い回しが巧妙」
→ 弁明しつつ朝霧投票に転じた。自分（花園）への疑惑を認めつつも朝霧を優先。流れに乗った印象。

**御影和也(mikage) 2周目**: 「朝霧・花園の1周目と2周目を通じた同調は偶然では説明不可能。2周目の同調は説明後に起きており独立判断の弁明と矛盾。巌は2周目で態度を変えたので朝霧・花園の仲間ではない示唆。朝霧に投票。朝霧が龍ヶ崎を急に最疑惑に格上げした後付け感も不自然」
→ 論理的に整理し朝霧投票宣言。自分（花園）への疑惑も維持しつつ朝霧を優先。

## 2周目終了後の状況分析
【投票予想】
- 朝霧投票確実: 霧島、天城、巌、御影（4票）
- 朝霧か花園: 猪熊（1票、どちらかに入る）
- 未表明: 白峰、龍ヶ崎
- 自分（花園）: 巌か龍ヶ崎に投票予定

朝霧さんが処刑される可能性が非常に高い。自分も疑われているが今日は朝霧さんが処刑されそう。

### 1日目 第1周

**白峰芽衣(mei)**: 「まだわからない、もう少し見ていないとなんとも言えない。変だと思ったら言う」
→ 様子見の無難な発言。初日としては普通。判断保留。

**龍ヶ崎大輔(ryugasaki)**: 「白峰の発言は情報量ゼロ。消極的な発言・様子見に疑いの目を向ける。発言の積極性を重視する」
→ 論理的・積極的姿勢。白峰に早速圧をかけている。積極性=善とは限らない。自分の判断軸を早々に宣言するのは人狼の誘導の可能性も。要観察。

**朝霧慎司(asagiri)**: 「龍ヶ崎の積極性重視に一理あると認めつつも白峰を庇う。論理だけでは見えない真実もあると龍ヶ崎を牽制」
→ 穏やかで観察的。龍ヶ崎の発言を分析して批評できている。バランスの取れた発言。今のところ村人らしい印象。

**猪熊太郎(inokuma)**: 「霧島の指摘面白い（朝霧・花園の同調）。龍ヶ崎の論理は悪くないが論理で場を仕切ることは目的を隠すためにも使える。全員を見ている」
→ 寡黙で観察的。龍ヶ崎への警戒も示す。自分も含めた「同調」を注目している。

**霧島彩音(kirishima)**: 「朝霧と花園が揃って白峰を擁護し龍ヶ崎を牽制したことに注目。方向性が似すぎている。白峰に『今変だと思う人はいるか』と質問」
→ 鋭い観察。自分（花園）が槍玉に挙げられている。ただし霧島の指摘自体は論理的に正当。「同調」を指摘するのは村人・人狼どちらもあり得る。白峰への質問で反応を引き出そうとしている点は村人らしい行動。

**天城りおん(amagi)**: 「朝霧と花園の発言方向が似すぎ。朝霧の言い方はフワッとして立場をぼかしている。朝霧に直接『今怪しいと思う人はいるか』と質問」
→ 感情的・直感的スタイル。自分（花園）と朝霧をセットで疑っている。ただし根拠は「モヤっとする」レベル。村人らしい直感型か人狼が誘導している可能性も。

**巌源蔵(iwao)**: 「朝霧・花園の同調は皆の指摘通り。だが霧島が初日から的確に場を読めるのは場慣れか情報を持っているか。賢すぎる奴には気をつけろ、と霧島を警戒」
→ 老練な視点。霧島への逆張り的疑惑。自分（花園）への直接言及は避け霧島に焦点を移した。村人らしい経験則的判断だが、霧島への疑惑は鋭い追及者を潰しにかかっている可能性も。要注目。

**御影和也(mikage)**: 「①朝霧と花園の同調は偶然とは考えにくい（龍ヶ崎批判・白峰擁護・雰囲気重視が同ベクトル）。②巌の霧島牽制は朝霧・花園への追及を分散させる効果がある。朝霧と花園が最有力、次いで巌を注視」
→ 最も整理された論理的発言。自分（花園）が最有力候補に名指しされた。巌への目配りも鋭い。御影は村人陣営の可能性高い。

## 推理メモ
【1周目終了時点の考察】
- 自分（花園）と朝霧への疑惑が集中。霧島・猪熊・天城・御影から注目されている。
- 巌は霧島への疑惑を出した。これが人狼の撹乱か正直な観察かは不明。
- 2周目で自分の立場を明確に説明する必要あり。「同調」は偶然だが独自の視点を示す。
- 朝霧への直接質問に朝霧がどう答えるかが重要。
- 龍ヶ崎は今のところ直接の疑惑がないが「積極性重視」の枠組みを最初に作った点は注視。

【2周目の発言方針】
- 「同調」について正直に説明する。朝霧の発言に共感したのは内容に共鳴したからで事前打ち合わせはない。
- 自分の独自観察を示す：巌の「霧島牽制」が追及分散になっている点に言及。
- 投票候補として巌か龍ヶ崎を視野に入れる。
